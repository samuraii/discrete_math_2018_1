## Перевод в 10 систему из любой, перевод из 10 системы в любую.

Для перевода в 10 систему выполнить:
```
1) 101101(2) -> ?(10)
2) 1 * 2^5 + 0 * 2^4 + 1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0 
3) 1 * 32 + 0 * 16 + 1 * 8 + 1 * 4 + 0 * 2 + 1 * 1
4) 32 + 0 + 8 + 4 + 0 + 1 = 45
```

```
1) 1073(8) -> ?(10)
2) 1 * 8^3 + 0 * 8^2 + 7 * 8^1 + 3 * 8^0 
3) 1 * 512 + 0 * 64 + 7 * 8 + 3 * 1
4) 512 + 0 + 56 + 3 = 572
```

```
1) 9AC(16) -> ?(10)
2) 9 * 16^2 + 10 * 16^1 + 12 * 16^0
3) 9 * 256 + 10 * 16 + 12 * 1
4) 2304 + 160 + 12 = 2476
```
