| A(2) -> A(16) | 0 | A(2) -> A(16) |
| :-----------: | :----------: |
| 0000 | 0      | 1000 | 8     |
| 0001 | 1      | 1001 | 9     |
| 0010 | 2      | 1010 | A     |
 
## Перевод в 10 систему из любой, перевод из 10 системы в любую.

Для перевода в 10 систему нужно выполнить:
```
1) 101101(2) -> ?(10)
2) 1 * 2^5 + 0 * 2^4 + 1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0 
3) 1 * 32 + 0 * 16 + 1 * 8 + 1 * 4 + 0 * 2 + 1 * 1
4) 32 + 0 + 8 + 4 + 0 + 1 = 45
```
